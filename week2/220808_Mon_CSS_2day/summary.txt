색상 처리(배경색, 글자색, 테두리 색 등)
    1)RGB : rgb(red, green, blue) 함수사용
        각 색상별 0~255 사이의 수치를  조합
        rgba(red, green, blue, alpha) 함수 - 투명도 반영
            alpha : 투명도(0.0 ~ 1.0) 0에 가까울수록 투명해짐
    2)HEX : 16진수로 색상을 지정
        예) #FF00FF : 보라색
            #abc -> #aabbcc 단축표현
            (ex)#ab3c=#aab3cc X 
                #fff=#ffffff O
                // 무조건 3개 다 같아야함)
            투명도 처리 : 8자리 수치로 표현   . 00: 0.0 , ff : 1.0
    3)HSL : hsl(hue, saturation, lightness) 함수 사용
        h - 색조, s - 채도, l - 명도
        H : 0 ~ 360 까지의 색상환 각도로 표현
        S : 0 ~ 100% 의 백분율 값, 0 - 회색음영, 100 - 색조의 본색
        L : 0 ~ 100% 의 백분율 값, 0 - 검은색, 100 - 흰색
        예) red - hsl(0, 100%, 50%)
        투명도 반영 : hsla 함수 사용

요소의 크기(너비와 높이) - 박스 모델(Box Model)
    문서를 구성하는 모든 요소는 box-model을 사용하여 크기와 여백을 처리

    일반적인 요소의 넓이 = width + (2*padding) + (2*border)
    예) width : 100px;
        padding : 10px;
        너비 : 20px
        w:100px p:;10px; 1px ->122px
    일반적인 요소의 높이 = height + (2*padding) + (2*border)

    요소의 크기에 바깥쪽 여백(margin)은 고려하지 않는다

    box-size 속성
    1) content-box : 기본 값, 일반적인 크기로 처리
        실제 너비/높이 = 너비/높이 + 패딩 + 테두리
    2) border-box : 길이(width/height)에 패딩과 테두리가 포함 
        실제 너비/높이 = 너비/높이

여백
    1) 안쪽 여백(padding) : 테두리의 안쪽, 요소의 컨텐츠 바깥쪽 영역
    2) 바깥쪽 여백(margin) : 테두리의 바깥쪽 영역

    padding
        1) padding - 모든 여백
        2) padding-top - 윗쪽 여백
        3) padding-right - 오른쪽 여백
        4) padding-bottom - 아랫쪽 여백
        5) padding-left - 왼쪽 여백
        padding 속성 하나로 4방향을 단축하여 사용 가능
        top을 시작으로 시계방향으로 나열 
        4개의 값을 지정 : top = right - bottom - left
        3개의 값을 지정 : top - right/left - bottom
        2개의 값을 지정 : top/bottom - right/left

        위/아래에 같은 값을 처리하고, 좌/우를 다른 값으로 처리하는
        방법은 없음 > 4개의 값을 지정해서 처리

        3방향은 같은 여백, 한 방향만 다른 여백을 줘야 하는 경우
        -> padding: 10px; padding-left: 30px;
        전체 여백 > 부분 여백 순으로 작성

    margin
        padding과 거의 동일
        margin auto : 요소의 가운데 정렬에 사용(세로 x)
        예) margin: 0 auto; (0은 수치가 들어가는 게 아니라 단위를 적지 않음)

CSS의 단위
    1) px(*) : 픽셀 단위. 1px = 1인치의 1/96
    2) pt : 포인트 단위. 1pt = 1인치의 1/76
    3) cm : 센티미터
    4) em : 출판에서 사용하는 문자의 가로길이(M 기준)
    5) 고정값 : thin(얇은), medium(중간), thick(두꺼운)
            -> 테두리에서 사용
    6) % : 백분율

테두리(border)
    요소의 테두리 선을 표현하는 속성
    여백과 마찬가지로 4방향에 따로 테두리를 설정할 수 있음
        1)border-style : 테두리의 종류(선의 모양). 실선, 점선, 파선 등
        2)border-width : 테두리의 두께
        3)border-color : 테두리의 색상
        4)border-radius : 모서리의 반지름(둥근 모서리 처리)
        5)border : 위의 각 속성을 합쳐서 사용하는 단축 속성
                두께 - 종류 - 색상 순으로 작성
                (최소 두께와 종류는 작성해야 테두리가 나타남)

minimum/maximum width/height
    1) min-width : 지정된 범위 이하로 줄지 않는 요소로 설정
    2) max-width : 지정된 범위 이상으로 커지지 않는 요소로 설정

배경(Background)
    요소의 배경 효과를 주기 위한 스타일 설정 속성
    1) background-color : 배경 색상을 지정하는 속성
    2) background-image : 배경에 이미지를 지정하는 속성
    3) background-repeat : 배경 이미지의 반복 여부 지정 속성
        repeat - x, y 반복(기본값)
        no-repeat - 반복 없음
        repeat-x - 가로 방향으로만 반복
        repeat-y - 세로 방향으로만 반복
    4)background-size : 배경 이미지의 크기 지정 속성
        주요 속성 값
        1> 수치(px, % 등)
        2> contain : 이미지가 요소 영역 내부에 모두 출력
                    공백이 발생할 수 있음
        3> cover : 이미지가 요소 영역을 덮음
                    너비 공백은 발생하지 않으나
                    높이 공백은 발생할 수 있음
    5)background-attachment : 배경 이미지의 스크롤 여부 결정
        속성 값 : fixed(고정, 스크롤되지 않음)
                  scroll(스크롤됨, 기본값)
    6)background-position : 배경 이미지의 위치 결정
        9곳의 위치를 지정
        1> left : 왼쪽 세로 중앙
        2> right : 오른쪽 세로 중앙
        3> top : 위쪽 가로 중앙
        4> bottom : 아래쪽 가로 중앙
        5> center : 정중앙
        6> left top/top left : 왼쪽 상단
        7> right top/top right : 오른쪽 상단
        8> left bottom/bottom left : 왼쪽 하단
        9> right bottom/bottom right : 오른쪽 하단
    7) background : 위 개별 속성을 하나로 처리하는 단축 속성
        크기(size)는 제외
        순서 : 색상 - 이미지 - 반복여부 - 스크롤여부 - 위치

텍스트 관련 속성
    컨텐츠(내용)에 들어가는 글자(하위요소 포함)에 대한 설정
    1) color : 글자색
    2) text-align : 정렬 방식. left(기본값), center, right, 
                    justify(양쪽 정렬/양쪽으로 맞춰줌)
    3) vertical-aligh : 수직 정렬 방식. 이미지 + 텍스트인 경우
        - top : 이미지와 글자를 윗선에 정렬
        - moddle : 가운데(중앙선)에 정렬
        - bottom : 밑선에 정렬(기본값)
    4) text-decoration : 글자 장식(선) 설정 및 제거
        - overline : 윗줄
        - underline : 밑줄
        - line-through : 취소선(삭제선). 가운데 줄
        - none : 줄 제거
    5) direction : 글자 인쇄 방향 설정, unicode-bidi와 함께 사용
        - ltr : 왼쪽에서 오른쪽으로 인쇄(기본값)
        - rtl : 오른쪽에서 왼쪽으로 인쇄
        - unicode-bidi 속성 : 유니코드의 방향 설정 보조 속성
    6) text-transform : 글자 변환(알파벳 기준)
        - uppercase : 소 > 대문자로 변환
        - lowercase : 대 > 소문자로 변환
        - capiralize : 단어의 첫글자를 대문자로 변환
    7) text-indent : 들여쓰기. 픽셀 단위로 들여쓰는 너비 지정
    8) letter-spacing : 글자 간격 지정
    9) word-spacing : 단어 간격 지정
    10) text-shadow : 글자에 그림자 효과를 ㅜ가
        설정 값 : 수평이동수치 - 수직이동수치 - 흐림수치 - 색상
    11) white-space : 공백 문자에 대한 처리
        normal : 일반 상태(기본값)
        nowrap : 연속 공백을 하나로 합침 <br>로 줄바꿈
        pre : 연속 공백 유지(편집화면대로) <br>로 줄바꿈
        pre-wrap : 연속 공백 유지. 개행문자(엔터)와 <br>로 줄바꿈
        pre-line : 연속 공백을 하나로 합침. 개행문자(엔터)와 <br>로 줄바꿈
    12) font-size : 글자의 크기를 지정
        사용단위 : px, em
